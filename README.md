# 🌌 Web Starter — создавайте приложения, просто описав их словами

Это стартовый набор, с которым **любой человек без знания программирования** может создать
своё веб-приложение или сайт — прямо в приложении **Claude Code Desktop**. Вы не пишете код.
Вы просто описываете, что хотите, отвечаете на пару простых вопросов — и приложение появляется.

> Внутри уже лежит красивая стартовая страница и «умные команды» для Claude. Дальше — только
> ваша фантазия.

---

## Что вы умеете делать сразу

| Команда | Что делает |
| --- | --- |
| `/start` | Запускает приложение и показывает его в окне справа. Начните с неё. |
| `/init-app` | Создаёт ваше приложение: задаёт вопросы, предлагает дизайн, собирает его. |
| `/build` | Продолжает сборку, если она прервалась. |
| `/push` | Сохраняет проект на GitHub в один шаг (commit + push). |

---

## Шаг 1. Установите две программы (один раз)

1. **Node.js** — это «движок», на котором работают сайты.
   Откройте [nodejs.org](https://nodejs.org), скачайте версию с пометкой **LTS** и установите
   (просто нажимайте «Далее»).
2. **Claude Code Desktop** — приложение, в котором вы будете общаться с Claude.
   Установите его и войдите в свой аккаунт.

> Проверять что-то в командной строке не нужно — Claude сделает это за вас.

## Шаг 2. Скачайте этот проект

На странице проекта в GitHub нажмите зелёную кнопку **Code → Download ZIP**. Распакуйте архив
в удобную папку, например `Документы/мой-проект`.

## Шаг 3. Откройте папку в Claude Code Desktop

Запустите Claude Code Desktop и откройте папку, которую только что распаковали
(**Open Folder / Открыть папку**).

## Шаг 4. Посмотрите демо

В поле чата напишите:

```
/start
```

Через минуту справа откроется красивая страница. Это значит, что всё работает 🎉

## Шаг 5. Создайте своё приложение

Напишите команду и кратко опишите свою идею, например:

```
/init-app сайт для записи клиентов в мою кофейню
```

Дальше Claude задаст несколько простых вопросов (нужно будет просто выбирать варианты),
**предложит на выбор несколько вариантов дизайна** под вашу идею и, когда вы одобрите план,
**соберёт приложение**. Чтобы снова его посмотреть — напишите `/start`.

> 🎨 **Дизайн.** Среди вариантов всегда есть пункт «Как на конкретном сайте» — выберите его и
> вставьте ссылку на сайт, который вам нравится. Claude возьмёт оттуда стиль (цвета, шрифты,
> настроение) и применит к вашему приложению. Чужие тексты и картинки не копируются — только
> внешний вид.

---

## Шаг 6. Сохраните проект на GitHub (когда захотите)

Когда приложение нравится и хочется его сохранить в облаке — напишите:

```
/push
```

При первом вызове Claude спросит, куда сохранять: создать новый репозиторий на GitHub или
подключить уже существующий (нужно будет вставить ссылку). Дальше откроется окно браузера для
входа в GitHub — это один раз. Все следующие `/push` будут просто сохранять изменения. Никаких
git-команд вводить не нужно. Ваши секретные ключи (`.env.local`) на GitHub **не попадают**.

## (Необязательно) Приложения с базой данных

Если вашему приложению нужно хранить данные (записи, заявки, личные кабинеты), Claude
предложит подключить **Supabase** — бесплатный облачный сервис баз данных. Когда дойдёт до
этого шага:

1. Заведите бесплатный аккаунт на [supabase.com](https://supabase.com).
2. Создайте токен доступа на
   [supabase.com/dashboard/account/tokens](https://supabase.com/dashboard/account/tokens).
3. Вставьте токен в файл `.env.local` как `SUPABASE_ACCESS_TOKEN` (Claude подскажет, как).

Если аккаунта пока нет — ничего страшного: Claude всё равно соберёт рабочую версию
приложения, а базу можно подключить позже.

## Частые вопросы

**Мне нужно знать код?** — Нет. Вы только описываете, что хотите, и выбираете ответы.

**Где «живёт» приложение?** — Пока на вашем компьютере (адрес `localhost:3000`). Публикацию в
интернете добавим позже.

**Что-то пошло не так.** — Просто напишите Claude, что случилось, обычными словами — он
поможет. Можно повторить команду `/start` или `/build`.

**Можно поменять уже готовое приложение?** — Да. Просто напишите, что хотите изменить
(«сделай кнопку зелёной», «добавь страницу с ценами») — Claude внесёт правки.
